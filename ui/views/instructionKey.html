Чтобы создать ключ доступа к Mistral AI:

  1. Cоздайте аккаунт Mistral или войдите в него на сайте console.mistral.ai.
  2. Перейдите на страницу "API Keys" – кнопка для перехода находится с левой стороны экрана
  3. Создайте новый API ключ, нажав оранжевую кнопку "Create new key"
  4. Подтвердите создание ключа, повторно нажав "Create new key" во всплывшем окне
  5. Скопируйте созданный ключ, нажав оранжевую кнопку "Copy key"
